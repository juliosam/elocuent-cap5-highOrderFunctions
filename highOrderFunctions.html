Funciones de Alto Orden

Las funciones de alto orden son aquellas que operan en otras funciones, ya sea utilizandola como argumento o regresandolas como resultado. Son muy utiles para trabajar con objetos y arrays extensos y ayudan a minimizar los bugs, haciendo mas clara la sintaxis

Con respecto a las funciones regulares, las funciones de alto orden toman un poco mas tiempo para computarse, pero la mayoria de las veces es insignificante en comparacion de las ventajas que dan para el trabajo de desarrollo

Entre este tipo de funciones se encuentra forEach, la cual recorre todo un arreglo aplicandole una callback function a cada uno de los elementos del arreglo.

Tambien existe map, que sonara parecido a forEach ya que tambien recorre el array aplicandole la callback a cada elemento, pero la diferencia es que nos regresa un arreglo con los resultados.

Luego esta some, la cual busca que se cumpla lo establecido en la callback function al menos en algun elemento en el array, devolviendo asi true y de lo contrario regresara false. Se le puede comparar con un or (||).

Tambien existe un equivalente a and (&&) que se llama every. El cual regresa true, solo si se cumple la callback en todos sus elementos.
